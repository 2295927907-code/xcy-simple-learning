# 光谱 SPD 自动计算器

## 启动方式

双击 `启动光谱SPD计算器.bat`，或在当前文件夹运行：

```powershell
.\启动光谱SPD计算器.bat
```

如果启动失败，当前文件夹会生成 `SPD_startup_log.txt`，里面会记录失败原因。

## 主要功能

- 自动读取当前文件夹中的 `AM1.5标准光谱数据280-4000nm.xlsx`。
- 自动读取当前文件夹中的 `1027SR.xlsx` 作为默认光谱响应 SR；也兼容旧的 `1025SR.xlsx`、`1025SR*.xlsx` 和文件名包含 `光谱响应` 的 SR 表格。
- 可选择参考光谱表格中识别到的光谱列作为标准；默认 AM1.5 文件会显示 `AM1.5G` / `AM1.5D`，选择 AM0 表格时会显示 `AM0`。
- 计算波段默认是 `300-1200 nm`，步长默认 `1 nm`。
- 可读取其他测试光谱表格：`.xlsx`、`.xlsm`、`.xls`、`.csv`、`.txt`、`.tsv`、`.asd`；文本光谱支持逗号、Tab、分号或空格分隔。
- 可手动选择测试表格的波长列、辐照度列和输入单位。
- 可通过 `批量导入光谱` 一次选择多个测试光谱文件，程序会自动识别波长列和辐照度列，并直接添加到 `光谱记录`。
- `光谱记录` 中会显示曲线样式；光谱名称可双击修改，图例和导出结果会同步使用修改后的名称。
- `光谱记录` 的 `独立显示` 列可单独控制每条测试光谱是否显示在界面曲线图中；点击 `批量显示` 或 `独立显示` 表头可弹出 `全选` / `全不选` 批量按钮，导出结果仍保留全部光谱。
- 选中 `光谱记录` 中的一行后，可用空格切换显示、`F2` 改名、`Enter` 高亮。
- 左上角 `恢复初始` 可清空当前导入和计算结果，界面恢复到刚打开时的默认状态。
- 可通过 `读取输入文件` / `保存输入文件` 读取或保存 `.inp` 输入状态文件；导出结果时也会同步保存同名 `.inp` 文件，方便下次恢复现场。
- 选择测试文件后，点击左上角 `开始计算` 得到结果。
- 输出 SPD 光谱偏差率、SPC 覆盖率、IEC 分段匹配等级、曲线图和数据预览。
- 多组光谱同时绘图时，程序会使用更丰富的颜色，并在颜色循环后用不同线型继续区分；图例会绘制在曲线右侧专用区域，光谱数量较多时图例字号会自动变小并分列显示。
- 双击图例中的测试光谱可多选高亮；被选曲线保持醒目，其余可见测试曲线和图例变淡，参考曲线保持不变。双击绘图区空白处可清除高亮。
- `ISC 修正` 页可读取光谱响应 SR，手动输入实测短路电流 ISC，或通过 `添加 IV` 一次读取一个或多个 IV 表格，并为每组电流指定测试光谱和温度；IV 支持 Excel、CSV、TXT、TSV 等格式，TXT 可读取空格分隔的电压/电流数据。
- ISC/CV 修正会计算 MG、MT、MMF 与修正后 CV，并按 A5 规则筛选有效 CV。
- 可导出 `.xlsx` 或 `.csv` 结果；`.xlsx` 包含 `Summary`、`Bands`、`Data`、`Charts`，如已计算 ISC 修正，还会包含 `CV_Groups`、`ISC_Correction` 和 `SR_Data`。
- 可单独导出 `光谱相关数据.xlsx`，`Summary` 页包含与界面一致的所有光谱对比图和汇总表，各光谱独立页面包含单组对比图、SPD、SPC、MG、MMF 和逐波长数据。

## 计算公式

SPD 按 IEC 60904-9:2020 语境下的光谱偏差率计算：

```text
SPD% = ∫ |E_test(λ) - E_ref(λ)| dλ / ∫ E_ref(λ) dλ × 100%
```

其中：

- `E_ref(λ)` 是当前在“参考光谱”中选择的标准光谱辐照度。
- `E_test(λ)` 是测试光谱辐照度。
- 界面不做归一化，直接用原始测试光谱计算 SPD。
- 测试光谱会线性插值到统一波长网格；未覆盖的波长按 0 处理。

SPC 按参考光谱覆盖率计算：

```text
SPC% = ∫ E_ref(λ) where E_test(λ) >= 0.1 × E_ref(λ) / ∫ E_ref(λ) × 100%
```

分段匹配使用 300-470、470-561、561-657、657-772、772-919、919-1200 nm 六个波段，并按每段测试占比 / 参考占比判断等级。

ISC 修正使用读取到的 SR 实际波段，并限制在 300-1200 nm 内；SR 没有覆盖到的波段不强行补齐：

```text
MG = ∫ E_ref(λ)dλ / ∫ E_test(λ)dλ
MT = 1 / (1 - 0.0004 × (25 - 测试温度))
MMF = (∫ E_test(λ)SR(λ)dλ / ∫ E_test(λ)dλ) / (∫ E_ref(λ)SR(λ)dλ / ∫ E_ref(λ)dλ)
CV = ISC × (MG × MT) / MMF
```

A5 筛选规则：单组 `|CVi - CVmean| / CVmean` 不超过 `1.5%`，ISC 范围 `(Imax - Imin) / Imean` 不超过 `1.5%`，CV 标准偏差 / CVmean 不超过 `1.0%`；超限时程序会剔除偏离最大的组并重复筛选，最终有效 CV 不少于 5 组。
